1. Simpan file login ke folder htdocs
2. Buat Database bernama login dan Import databases login yang sudah disediakan di Folder "DATABASE" kedalam database login yang telah dibuat di phpmyadmin
3. Setelah import databases login ke phpmyadmin maka bisa masuk ke http://localhost/login/
4. Agar bisa masuk bisa dafar menjadi member menggunakan fitur create account.
5. Lalu bisa masuk sebagai member.
6. Ketika ingin login sebagai admin maka bisa login menggunakan
email : admin@gmail.com
password : 12345
7. Setelah login menggunakan akun admin maka dapat login sebagai admin dan menggunkaan fitur yang dapat diaccess sebagai admin
